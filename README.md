# totsugen

![Build Status](https://github.com/gotoeveryone/totsugen/workflows/Build/badge.svg)

## Requirements

- Go 1.17+

## Run

```console
$ go run src/main.go -value {value} # ex: go run src/main.go -value hogefuga
```

## Format check

```console
$ go vet -v ./...
```

## Test

```console
$ go test -v ./...
```
